# 📊 Customer Churn Prediction & Dashboard

An end-to-end Machine Learning solution designed to predict customer churn and provide data-driven insights through an interactive web dashboard. Built using Python, Scikit-Learn, and Streamlit.

## 🚀 Features
- **Predictive Modeling:** Uses a Random Forest Classifier trained on key demographics to compute churn risk.
- **Interactive Analytics:** Rich visualizations displaying correlations between customer tenure and churn trends.
- **Real-Time Testing:** Sidebar entry allowing users to pass custom customer metrics to fetch instant churn probabilities.

## 🛠️ Installation & Setup
1. Install dependencies: `pip install -r requirements.txt`
2. Train the ML Model: `python src/model.py`
3. Launch the Dashboard UI: `streamlit run src/app.py`

## 🧮 Tech Stack Used
- Python, Pandas, Scikit-Learn, NumPy, Joblib, Streamlit, Plotly Express
